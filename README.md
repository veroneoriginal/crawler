Реализовать асинхронный краулер для новостного сайта news.ycombinator.com.
Краулер:
1) запускается каждые N секунд,
2) парсит топ новостей, сохраняя каждую, 
3) а также сохраняет все ссылки из обсуждения конкретной новости.

Установить библиотеки
pip install aiohttp beautifulsoup4

Запуск python main.py
